Creating Boards

Creating a board is simple. Once you have an account and you are signed into the Trello website, click the boards link in the top left corner and select "Create new board..."

All you then have to do is name your board and select the privacy level:

  • Private - Only people specifically added to this board can view and edit it
  • Organisation - Visibile to members of the organisation. Only members added to the board can edit.
  • Public - Visible to anyone with the link and will show up in search engines. Only people added to the board can edit

Once the board is created you can add members who can contribute to the board. This is done by clicking the Add members icon found on the side menu.

The boards are organised into lists. These can be thought of as the different swimlanes on our existing kanban boards and can be added by clicking "Add a list..."

Once the list have been setup you can start adding stories and tasks.
