Trello

Chris Gathercole/Mat Hayes - Learning Plan 3


Introduction

Trello is a free to use tool that offers users the option to create individual and collaborative work boards.

Trello can be used within web browsers or with Apple and Android devices.

When working collaboratively, work boards update real time and allow users to assign themselves to tasks.

Story progress is tracked as a percentage as associated tasks are completed.

Deadlines and notes can be recorded against tasks.

Topics Covered

  • Creating Boards
  • Adding Stories and Tasks
