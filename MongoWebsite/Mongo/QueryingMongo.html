Querying Mongo



show dbs List the databaes on the mongo instance
db.createCollection("< some DB name >") Creates a database
use <dbname> will switch to the specified database
db will show the currently connected database
show collections will show the collections available in the connected database
db.<Collection Name>.find() will query the specified collection for all its records
