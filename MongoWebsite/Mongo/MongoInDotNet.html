Initialising Mongo in .Net

This section looks at using Mongo within .net applications.

In order to use Mongo Nuget will need to be used to get the Mongo .Net drivers and add a reference to the project. The key objects part of the .Net Mongo driver are:

  • MongoClient is the root object to handle connecting to the database
  • MongoServer gets an instance of the MongoDB server
  • MongoDatabase represents a database on a MongoDB server
  • MongoCollection<T> represents a collection in a MongoDB database

The following code snippet will establish a connection pool to the "CMdb" database leaving it ready to start pulling data out of collections. If the database does not exist then it will be created on first use (same applies for the collection):

    public class Carer
    {
        private MongoClient _mongo;
        private MongoServer _server;
        private MongoDatabase _database;
        private MongoCollection<carerobj> _carers;
        
        public Carer()
        {
            _mongo = new MongoClient();
            _server = _mongo.GetServer();
            _database = _server.GetDatabase("CMdb");
            _carers = _database.GetCollection<carerobj> ("carers");
        }
    }
